The default prefix of Tyra is ">".

This bot aims to replace all the others in order to limit the case: "members: 5", "bots: 8".

We use Node.js version 12 minimum for our robot in order to be able to freely run the discord.js version 12 library.

Moderation Commands

Our bot has a manual moderation system to make it easier for you against disrespectful members.

Warn, Unwarn and infractions

You can notify a member with the command: [prefix]warn [user] [reason]. You can also remove a warning with the command: [prefix]unwarn [user/idwarn] tandit that the command to see the number of infringements will be available during v2.

Mute, Unmute and Tempmute

You can mute a member (can no longer send a message) for an indefinite period of time with the command: [prefix]mute [user] [reason] and give him back his voice with the command: [prefix]unmute [user]. You can also mute a member for a specified period of time with the command: [prefix]tempmute [user/duration] [duration/user].

Kick, Ban and Tempban

You can exclude a member (can return) with the command: [prefix]kick [user] [reason] definitively exclude it with the command: [prefix]ban [user] [reason]. You can also ban for a fixed period with the command: [prefix]tempban [user/duration] [duration/user].
